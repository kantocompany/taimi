Key observations

  • GitHub Copilot Pro ($10/mo) is the cheapest paid individual plan. Mistral Vibe ($14.99/mo) is the only EU-headquartered vendor — critical for data sovereignty.
  • Team seat costs range 10× from $19 (Copilot Business) to $200 (Augment Max).
  • Usage-based is apples to oranges: Copilot charges per request ($0.04), Anthropic/OpenAI/Mistral per token, Devin per ACU, Replit per effort-based Agent task. Cursor dropped explicit overage — usage scales with plan tier (3×/20×).
  • Mistral's Devstral 2 output tokens ($2.00/M) are ~12× cheaper than Claude Opus 4.6 output ($25/M). Sonnet 4.6 ($15/M) sits between.
  • Platform bundlingP: Claude Code, Codex, and Vibe subscription plans include their full platforms (Claude.ai, ChatGPT, Le Chat). Their prices aren't directly comparable to standalone tools like Cursor, Windsurf, or Gemini Code Assist (free individual tier, 6K completions/day).
  • Self-hosted / on-prem: Windsurf (FedRAMP), Mistral (EU data, fine-tune), Cline and Aider (open-source BYOK).
